# API DO APLICATIVO PARA PRÁTICA DE DIVERSOS IDIOMAS

## TENCNOLOGIAS

 - ADONIS5
 - TYPESCRIPT
 
### Instalação 
  
  Após clonar o repositório você deve rodar o seguinte comando.

 ```shell
   cd smarthidiomas-backend

   npm install
   
   node ace server --watch
 ```
